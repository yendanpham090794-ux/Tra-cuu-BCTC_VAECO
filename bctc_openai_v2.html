☁️ Google Drive — Đồng bộ Tài liệu

Lưu thư viện tài liệu lên Google Drive để dùng được trên mọi thiết bị. App sẽ đọc/ghi một file bctc_library.json trong Drive của bạn.

1
Vào console.cloud.google.com → Tạo project → Bật Google Drive API
2
Tạo OAuth 2.0 Client ID (loại: Web application) → Thêm http://localhost và URL file này vào Authorized origins
3
Dán Client ID vào bên dưới và nhấn Kết nối
